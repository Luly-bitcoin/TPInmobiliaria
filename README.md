# TP Inmobiliaria

Aplicación móvil Android desarrollada en Java para la gestión de inmuebles, propietarios y contratos.

## Características

- Login de propietarios
- Visualización de inmuebles
- Detalle de cada inmueble
- Cambio de disponibilidad
- Perfil del propietario
- Cambio de contraseña
- Integración con API REST
- Google Maps con vista híbrida y efecto 3D

## Arquitectura

El proyecto utiliza:

- Fragments
- ViewModel
- LiveData
- RecyclerView Adapter
- Retrofit para consumo de API REST

### Inmuebles
- Listado de propiedades
- Estado disponible/no disponible
- Detalle completo del inmueble

### Contratos
(Aun en proceso)

### Perfil
- Edición de datos del propietario

### Mapas

- Vista híbrida con inclinación 3D
